The Online Examination System is a web-based application designed to modernize the way academic assessments are conducted. It provides a secure, user-friendly interface for students to take exams remotely, featuring real-time tracking and automated submission logic.
Key Features
• Student Dashboard: A centralized hub where students can view active exams and check their previous results.
• Real-time Countdown Timer: An integrated JavaScript timer that alerts the user when time is running low and automatically submits the exam once the duration expires.
• Interactive Quiz UI: A clean, focused interface for multiple-choice questions (MCQs) that prevents distractions.
• Responsive Design: Fully optimized for different screen sizes, ensuring accessibility on mobile devices, tablets, and laptops.
Technical Specifications
• Frontend: HTML5 and CSS3 (Bootstrap framework used for the dashboard layout and responsive grid system).
• Logic: JavaScript for handling the countdown timer functionality and form validation.
• Development Tools: Developed using VS Code and managed via GitHub (Asiviguy profile).
Objective
The primary goal of this project was to demonstrate proficiency in frontend development and client-side scripting as part of the BCA 4th Semester curriculum. It focuses on creating a functional, time-sensitive application that mimics real-world e-learning platforms.
